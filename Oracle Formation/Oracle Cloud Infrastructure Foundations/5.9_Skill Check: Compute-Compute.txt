1. What is the primary purpose of Oracle Cloud Infrastructure Functions?
O To execute code in response to events or HTTP requests
O To provide a managed database service
O To deploy and manage virtual machines
O To store and manage files
Answer: To execute code in response to events or HTTP requests

2. Which two parameters can be customized when creating a flexible shape compute instance?
[] Number of physical NICs
[] Number of virtual NICs
[x] Number of OCPUs
[x] Amount of memory
Answer: Number of OCPUs, Amount of memory

3. Which type of storage is associated with instances in the OCI Compute service?
O Archive Storage
O Object Storage
O Block Storage
O File Storage
Answer: Block Storage

4. Which statement about the working of autoscaling in an instance pool is true?
O It automatically provisions and removes instances in an instance pool.
O It automatically changes the shape of the compute instance to increase the number of OCPU count and memory.
O It can perform only schedule-based autoscaling.
O It can perform only metric-based autoscaling.
Answer: It automatically provisions and removes instances in an instance pool.

5. Which processor type is NOT available for the OCI Compute service?
O AMD
O Ampere
O Snapdragon
O Intel
Answer: Snapdragon
